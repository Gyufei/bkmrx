bkmrx HTTP API

监听 http://127.0.0.1:8733，无鉴权，仅供本机 App 与浏览器扩展使用。

数据模型

{
  "id": 42,
  "url": "https://example.com",
  "title": "Example",
  "description": "",
  "tags": ["rust", "中文"],
  "access_count": 3,
  "created_at": "2026-07-23T12:00:00Z",
  "updated_at": "2026-07-23T12:10:00Z",
  "accessed_at": null
}

路由

GET /api/health
GET /api/bookmarks?query=&tags=rust,中文&cursor=&page_size=50
POST /api/bookmarks
GET /api/bookmarks/by-url?url=https%3A%2F%2Fexample.com
GET /api/bookmarks/:id
PATCH /api/bookmarks/:id
DELETE /api/bookmarks/:id
GET /api/tags

创建

{
  "url": "https://example.com",
  "title": "Example",
  "description": "",
  "tags": ["rust", "中文"]
}

成功返回 201 和完整 Bookmark。PATCH 接受相同字段的任意子集并返回 200；DELETE 成功返回 204。

分页

page_size 默认为 50，范围为 1–100。标签以逗号分隔且必须全部命中。后续请求应原样传回不透明的 next_cursor。

{
  "items": [],
  "next_cursor": null
}

{
  "error": {
    "code": "bookmark_url_conflict",
    "message": "A bookmark with this URL already exists",
    "details": { "url": "https://example.com" }
  }
}
