bkmr-desktop API

HTTP service running on 127.0.0.1:8733

Endpoints

GET /api/tags

获取所有标签及其数量，用于 Chrome 插件中的分类选择。

Response 200 OK

[
   { "name": "fe", "count": 852 },
   { "name": "全栈", "count": 323 }
 ]
POST /api/bookmarks

添加书签到 bkmr 数据库。

Request Body

{
   "url": "https://example.com",
   "title": "Example Title",
   "tags": ["dev", "rust"]
 }
字段类型必需说明
urlstring是书签 URL
titlestring否书签标题，缺省则使用 URL
tagsstring[]否标签列表，缺省为空

Response 201 Created

{ "id": 1234, "status": "created" }

Error Response

{ "error": "bkmr add failed: ..." }

Authentication: None. Listen-only on 127.0.0.1.
